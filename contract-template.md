# Contract Template

The contract template now lives as a skill, so it travels with the plugin:
**[`skills/shared/contract-template/SKILL.md`](skills/shared/contract-template/SKILL.md)**.

The Planner loads it through its `contract-template` skill. See that file for the
tier-sized template (`[core]` for a Standard task, `[full]` added for Domain) and the
immutability rules.
